============
Installation
============


Stable release
--------------

To install dragon-axe, run this command in your terminal:

.. code-block:: console

    $ pip install dragon_axe

This is the preferred method to install dragon-axe, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for dragon-axe can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/dafu2/dragon_axe

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/dafu2/dragon_axe/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/dafu2/dragon_axe
.. _tarball: https://github.com/dafu2/dragon_axe/tarball/master
